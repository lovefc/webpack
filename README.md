## 一个小项目

关于fcup.js如何在node中用webpack打包,和后端处理

正好用来学习一下webpack打包


## 本地使用

```bash
# 克隆本仓库
$ git clone https://gitee.com/lovefc/webpack.git

# 安装依赖
$ npm install webpack -g //直接安装新版即可
$ cd webpack  //进入目录
$ npm install

# 编译项目
$ npm run watch (webpack --watch) //或者使用 webpack -p

# 启动开发服务器
$ npm run dev

```
